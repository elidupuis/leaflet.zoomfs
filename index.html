leaflet.zoomfs

An extension of the default Leaflet zoom control to include a fullscreen button.

Get all the details at Github.

Dependencies

Leaflet (https://github.com/CloudMade/Leaflet)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/elidupuis/leaflet.zoomfs

get the source code on GitHub : elidupuis/leaflet.zoomfs
